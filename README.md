# API Project
Accounts Transfer API

# Dependencies

- Java 8
- Maven 3.x
- Dropwizard
- JUnit
- JavaHamcrest

# API Endpoints

<pre>
- GET   /accounts
- POST  /accounts
- GET   /accounts/{id}
- POST  /accounts/{id}/deposit
- GET   /transactions
- GET   /transactions/{id}
- POST  /transactions/transfer
</pre>
